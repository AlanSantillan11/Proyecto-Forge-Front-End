# 🚀 Proyecto Forge – Front End

Este repo reúne todos los trabajos prácticos que fui haciendo durante el curso de **Front End en Forge Bootcamp**.  
A lo largo de este proceso fui aprendiendo y aplicando tecnologías como **HTML, CSS, JavaScript y React**, además de herramientas como **Vite** y **Bootstrap**.

Cada carpeta representa una práctica distinta o un mini proyecto que me ayudó a entender mejor los conceptos del desarrollo front end: desde eventos básicos en JavaScript hasta manejo de estados y rutas con React.

---

## 🧠 Tecnologías que usé

- HTML / CSS
- JavaScript
- React.js
- Vite
- Bootstrap
- Git y GitHub

---

## 📂 Qué vas a encontrar

| Carpeta | Descripción |
| --- | --- |
| `EstadosCore` | Ejercicios sobre manejo de estado y renderizado condicional. |
| `FormulariosCore` | Prácticas con formularios controlados y validaciones. |
| `ImagenesCore` | Carga dinámica de imágenes y componentes visuales. |
| `NotasCore` | App de notas en React, con manejo de componentes y props. |
| `RouteoCore` | Ejemplo con React Router y navegación entre vistas. |
| `examen-cinturon1` | Proyecto integrador del curso. |
| `FrontEnd-ProyectoFinal` | 🎓 Proyecto final del curso — app fullstack con React, Node.js, Express y MongoDB. Incluye autenticación, gestión de emergencias y roles de usuario (voluntarios y organizaciones). |
| Otras carpetas | Ejercicios más chicos como eventos, botones, diseño de tarjetas, etc. |

## ⚙️ Cómo probar los proyectos

Si alguno de estos proyectos tiene un `package.json`, podés levantarlo así:

```bash
npm install
npm run dev
```

> Para el proyecto final (`FrontEnd-ProyectoFinal`), hay que levantar tanto el `cliente` como el `servidor` por separado, cada uno con sus propios comandos.
